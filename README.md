# Hungry-Hiss
Classic Snake Game
  Hungry Hiss is a classic snake game developed using Java Swing and AWT libraries. It includes a graphical user interface featuring a snake that moves around the screen, eats food, and grows in length. The game also includes obstacles that the player must avoid, and a scoring system based on the length of the snake. The application includes keyboard controls for directional movements, as well as error handling for invalid inputs.
